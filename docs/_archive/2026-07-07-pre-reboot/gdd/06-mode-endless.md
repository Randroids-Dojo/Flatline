# Mode: Endless Room

**Status:** done

The player survives as long as possible in a single room. The run ends when health reaches zero.

Scoring rewards:

- Kills
- Survival time
- Combos
- Accuracy
- Close-range kills
- Kill variety
- No-damage streaks
- Resource efficiency
