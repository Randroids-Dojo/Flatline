# Relics (one-run contraband)

**Status:** done

Bought at The Fence after unlocking it on the board. Each applies to the next run only and is consumed when the run starts. One of each per run.

- Rabbit's Foot 250: survive one death at 50 hp.
- Double Espresso 150: +25% move speed.
- Picnic Basket 120: start 50 hp over the cap.
- Loaded Dice 200: double coin drops.
- Iron Umbrella 180: start with 100 trench armor.
- Case of Ammo 160: start with full pockets.
- Skeleton Key 220: vault doors open without keys.
- Bloodhound Nose 100: pickups appear on the automap.
